Прочитать и/или законспектировать лекцию. Кому надо конспектировать: переписать ВЕСЬ представленный в лекции код + теорию (достаточно кратко термины и особенности).

1. Доделать семантическую вёрстку сайта как на картинке. Проверять валидацию через сайт https://validator.w3.org/
2. Создать в папке CSS файл styles.css и определить все необходимые стили. Создайте эффекты при наведении мыши на пункт меню в шапке сайта. Проверить валидацию через сайт https://jigsaw.w3.org/css-validator/
3. В CSS и HTML файлах при валидации не должно быть никаких ошибок, предупреждений и т.д. Т.е. валидация должна быть без сообщений.
4. Подключить стили к страничке, чтоб ваша страничка выглядела как на картинке.
5. Для себя можно поиграть в игры для закрепления знаний по CSS:
https://flexboxfroggy.com/#ru
https://cssgridgarden.com/#ru
6. Продемонстрировать готовый сайт преподавателю на занятии.
